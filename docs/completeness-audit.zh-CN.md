# v1.0 完整性审计（Completeness Audit）

本文件用于说明 v1.0 当前已经完成的功能范围，以及明确不会做的内容。

---

## 已完成（v1.0）

- 文献监控（NeuroPET / MRI-to-PET / Tau PET / AD）
- 医学 VLM / CLIP / foundation model 更新监控
- GitHub / HuggingFace 更新追踪
- 7 天 + 30 天 fallback 检索
- 标题 + 链接核验
- 去重（DOI / arXiv / URL / 标题）
- 排序评分
- 中文六段式周报
- GitHub Actions 自动运行
- SMTP 邮件发送

---

- MCP server
- Web UI
- 浏览器插件
- 自动下载论文全文
- 绕过网站访问限制
